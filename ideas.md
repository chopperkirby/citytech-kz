<idea>
**Design Movement**: Kaspi-Inspired Civic Minimalism
**Core Principles**:
1. Trust through clarity — every element earns its space
2. Data-forward hierarchy — numbers and status are the heroes
3. Mobile-first touch targets — thumb-friendly, no clutter
4. Institutional confidence — feels like a government tool that actually works

**Color Philosophy**: Deep navy (#0A2463) as the anchor of authority, electric blue (#1E88E5) for action states, white (#FFFFFF) as breathing space. Accent amber (#F59E0B) for warnings, red (#EF4444) for critical, green (#22C55E) for community. The palette communicates urgency gradients without aggression.

**Layout Paradigm**: Asymmetric card-stack layout. The feed uses a left-aligned list with a sticky right-panel for the map. On mobile, tabs switch between feed and map. The auth screen uses a split: left is a branded hero panel, right is the form.

**Signature Elements**:
1. Priority score "badge" — a circular progress ring around issue cards
2. Category color-coded left border on each issue card (red/yellow/green stripe)
3. Floating "Подать заявку" button with a subtle pulse animation

**Interaction Philosophy**: Instant feedback. Every tap/click triggers a micro-animation. Upvote button morphs on press. Map clusters explode into lists with a slide-in panel.

**Animation**: Entrance animations use staggered fade-up (50ms delay per card). Upvote uses a spring-scale effect. Map side panel slides in from the right. Auth form fields animate in sequence.

**Typography System**: "Manrope" (700/800) for headings — geometric, authoritative. "Inter" (400/500) for body text. Russian/Kazakh text renders cleanly in both.
</idea>

<idea>
**Design Movement**: Soviet Constructivism Reborn
**Core Principles**:
1. Bold geometry — rectangles, hard edges, no rounded corners
2. High contrast — black/white with single accent color
3. Typography as structure — large text blocks define layout zones
4. Functional brutalism — raw utility elevated by intentional composition

**Color Philosophy**: Off-white (#F5F0E8) background, near-black (#1A1A1A) text, single accent of Kazakhstan flag blue (#00AFCA). Status colors are pure: red, yellow, green with no tinting.

**Layout Paradigm**: Grid-based with visible column lines. Issue cards are newspaper-style rows. The map takes up a full-width strip. Auth screen is full-bleed with oversized typography.

**Signature Elements**:
1. Thick horizontal rules separating sections
2. Issue numbers displayed in massive 72px type
3. Status indicators as solid color squares, not circles

**Interaction Philosophy**: No animation — instant state changes. Hover states use color inversion. Brutally honest UI.

**Animation**: Minimal — only essential transitions (100ms opacity). No spring physics, no parallax.

**Typography System**: "Space Grotesk" (900) for display. "IBM Plex Mono" for data/numbers. Creates a technical, precise aesthetic.
</idea>
